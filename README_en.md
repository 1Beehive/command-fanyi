## The translation is done by Baidu Translate. You can change it to other translations.
> Set the appid and app secret of Baidu Translate
>
> export APPID="**********"
>
> export APPSECRET="**********"

## Install
```shell
brew install fanyi
```

## Example
```shell
go mod -h fanyi
tldr git push fanyi
tldr curl fanyi
```
